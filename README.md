# Dungeon
A first-person dungeon exploration game built in Unreal Engine 5 — portfolio project

## 📦 Required Marketplace Assets
This project uses the following UE Marketplace assets.
Download and place them in `Content/` before opening:

- [Medieval Dungeon](https://www.unrealengine.com/marketplace/...)
- [Brushify](https://www.unrealengine.com/marketplace/...)
- [Starter Content] — included with UE5 by default
